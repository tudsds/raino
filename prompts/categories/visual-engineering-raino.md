# Visual Engineering — Raino

Creates the hacker/cyberpunk design system and both Raino websites.

## Design Language

- Dark-first (near-black backgrounds, neon accents)
- Monospace fonts for technical content
- Gradient accents (cyan → purple → magenta)
- Grid/circuit-board patterns as background elements
- High contrast for legibility
- Glowing borders and hover effects
- Sharp corners mixed with subtle radius
- Responsive layouts

## Color Palette

- Background: #0a0a0f, #111118, #1a1a24
- Primary: #00f0ff (cyan)
- Secondary: #8b5cf6 (purple)
- Accent: #ff00aa (magenta/pink)
- Success: #00ff88 (green)
- Warning: #ffaa00 (amber)
- Error: #ff3366 (red)
- Text: #e4e4e7 (primary), #71717a (muted)
- Border: #27272a, #3f3f46

## Typography

- Headings: Space Grotesk or similar
- Body: Inter or similar
- Code/technical: JetBrains Mono or similar

## Components Needed

Button, Card, Panel, Badge, Input, Select, Textarea, Modal, Tabs, Progress, StatusDot, NeonBorder, GlowingCard, CircuitGrid
