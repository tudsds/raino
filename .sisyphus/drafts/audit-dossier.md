### CRITICAL (Must Fix Before Launch)

1. **Project page ownership bypass** — `projects/[id]/page.tsx` has no ownership check
2. **No Prisma migrations** — Fresh deployment has no schema
3. **No design job dispatch** — Jobs queue forever, nothing processes them
4. **No real embedding provider** — RAG returns hash-based nonsense vectors
5. **$0 price wins best-price** — null→0 bug corrupts quote comparison
6. **Artifacts ephemeral** — Written to /tmp/, lost on restart

7. Auth callback silently swallows provisioning errors

9. KiCad CLI silently falls back to fixture mode without error
10. Language switcher is fake (external links only)
11. NEXT_PUBLIC_APP_URL localhost fallback risk
12. pgvector dimension 384 → needs 1536 for real embeddings
13. No Zod validation on supplier API responses
14. No studio E2E tests (Playwright covers site only)
